# HalalChecker AI
Are you a Muslim who have been wanting to search for a specific item but don't know if it is Halal or not? That's where HalalChecker AI comes in! It uses Gemini to figure out if a given snack is halal or not and it would help you make good choices. 

## Features
- The user can type any food brand, snack, or a food resturant to determine if the given query is Halal or not

## Tech Stack
Backend: Python, Flask<br>
Databases: SQL, CRUD <br>
Frontend: React, Vite, Tailwind.css, Progressive Web Apps, CSS, JavaScript
